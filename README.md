# LearnMySQL

本项目记录自己在学习MySQL的过程中遇到的一些优秀项目。包括一些MySQL的学习资源和一些优秀的Go项目代码。尽可能的理解MySQL，尽可能的使用MySQL。作者：[0e0w](https://github.com/0e0w/LearnMySQL)

本项目创建于2020年10月12日，最近一次更新时间为10月12日。
